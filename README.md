# AI Analytics Dashboard

A Next.js analytics dashboard for tracking model usage, request metrics, and billing activity.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Recharts

## Local Development

1. Install dependencies:
   - `npm install`
2. Start the app:
   - `npm run dev`
3. Open:
   - `http://localhost:3000`

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run lint checks

## Project Structure

- `app` - Routing, layout, global styles
- `components` - Dashboard UI and page components
- `lib` - Shared utilities and configs
- `public` - Static assets

## Configuration

Copy `.env.example` to `.env.local` and set any required keys for external services.
